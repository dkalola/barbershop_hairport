/appointments?apiKey=[YOUR API] - Get all appointments

 &guestId=[Guest ID]

  Returns an array of appointments that matches Guest ID.

  NOTE: Cannot be used with appointment ID Filter options.


 &appointmentId=[appointment ID]

  Returns the appointment by appointment ID.

  NOTE: Cannot be used with Guest ID Filter options.


 &time=[true/false]&guestId=[Guest ID]

  Returns an array of upcoming appointments that matches Guest ID.

  NOTE: Cannot be used with appointment ID Filter options.


 Data Format

                  
                      {
                          "data": [
                              {
                                  "checkedIn": [type-Boolean],
                                  "amount": [type-Number],
                                  "time": {
                                      "_seconds": [type-Number],
                                      "_nanoseconds": [type-Number]
                                  },
                                  "_id": [type-String],
                                  "date": {
                                      "_seconds": [type-Number],
                                      "_nanoseconds": [type-Number]
                                  },
                                  "payed": [type-Boolean],
                                  "guestID": [type-String],
                                  "location": [type-String]
                              }
                          ]
                      }

/guests?apiKey=[YOUR API] - Get all Guests

 &guestId=[Guest ID]

  Find Guest by Guest's ID

  NOTE: Cannot be used with email ID Filter options.


 &email=[email ID]

  Find Guest by Guest's email ID

  NOTE: Cannot be used with Guest's ID Filter options.


 Data Format

                  
                      {
                          "data": [
                              {
                                "_id": [type-String],
                                "name": [type-String],
                                "email": [type-String],
                                "date": {
                                    "_seconds": [type-Number],
                                    "_nanoseconds": [type-Number]
                                },
                                "phone": [type-String]
                            },
                          ]
                      }
